# DSA-Basics--C
Data Structures and Algorithms implemented in C- This repository contains a collection of fundamental DSA concepts
#1. Sorting Algorithms
Bubble Sort: Basic comparison-based sorting.
Selection Sort: In-place comparison sort.
Insertion Sort: Efficient for small data sets.

Merge Sort: Divide and conquer approach.
Quick Sort: Efficient partitioning-based sort.

#2. Linked Lists
Singly Linked List: Basic node-to-node traversal.
Doubly Linked List: Bi-directional traversal.
Circular Linked List: Last node points back to the head.

To compile any of the files, use a C compiler like gcc:
gcc filename.c -o output
./output
